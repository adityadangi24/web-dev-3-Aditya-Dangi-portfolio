Aditya Dangi Portfolio Website
Lab Title:
One-Page Responsive Portfolio Website with Flexbox, Grid, Media Queries, and Animations

Description:
This project is a one-page responsive portfolio website designed and built using HTML, CSS, Flexbox, CSS Grid, media queries, and animations. The portfolio showcases the personal information, skills, and projects of Asmit, a Full Stack Developer.

The website includes interactive features like hover effects, smooth transitions, and animated text to create a polished and engaging user experience. The layout adapts seamlessly across desktop, tablet, and mobile screens using a mobile-first approach with relative units such as %, rem, vw, vh for scalable spacing and font sizes.

Key Features:
Flexbox: Used in the header and hero sections for alignment and spacing
CSS Grid: Applied in Skills and Projects sections for structured layouts
Responsive Design: Mobile-first design, works well on desktop, tablet, and mobile
Animations: Color-changing role in hero section and fade-in effect on image
Hover Effects: Buttons, navigation links, skill cards, and project cards
Interactive Contact Form: Name, Email, and Message fields
Project Structure:
portfolio-website/ │ ├── index.html # Main HTML file ├── style.css # CSS styles including Flexbox, Grid, animations, and responsiveness ├── image.jpg # Profile photo └── README.md # Project description and instructions

Steps to View Responsive Behavior:
Open the index.html file in any modern browser (Chrome, Firefox, Edge).
Resize the browser window or use device simulation to test responsiveness.
Breakpoints Tested:
Desktop: 1024px and above – Hero text on the left, profile image on the right
Tablet: 768px – Hero section stacks vertically, image and text centered
Mobile: 500px and below – Full stacking layout, text and buttons scaled for readability
Interact with buttons, links, and project cards to see hover effects and animations.
Observe the color-changing “Full Stack Developer” text in the hero section.
Technologies Used:
HTML5
CSS3 (Flexbox, Grid, Transitions, Animations)
Relative units (%, rem, vw, vh) for responsiveness
